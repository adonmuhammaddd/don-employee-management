
## Installation

Clone this root project

### Front End

Execute from root project folder
```bash
  cd don-employee-nextjs
  npm install
  npm run dev
```
### Database
Create database `employee_db`
```bash
  CREATE DATABASE IF NOT EXISTS `employee_db`
```
### Back End
Execute from root project folder
```bash
  cd don-employee-nestjs
  npm install
  npm run seed
  npm run start:dev
```
